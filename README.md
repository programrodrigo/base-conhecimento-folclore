<h1>Base de Conhecimento do Folclore Brasileiro</h1>
<p>Projeto de cunho pedagógico em HTML, CSS e JavaScript</p>

<h2>Sobre o Projeto</h2>
      <p>
        Este projeto reúne lendas do folclore brasileiro em uma base digital construída com 
        <strong>HTML, CSS e JavaScript</strong>. O sistema permite que o usuário digite termos e receba como resultado 
        lendas relacionadas, acompanhadas de nome, região e breve descrição.
      </p>
      <p>
        Mais do que um recurso tecnológico, o projeto valoriza a cultura popular e ajuda a manter vivas histórias que 
        fazem parte da identidade do Brasil. Ele pode ser usado em escolas de Educação Infantil e Ensino Fundamental 
        como ferramenta pedagógica, despertando a curiosidade das crianças e incentivando a leitura.
      </p>

<h2>Objetivos</h2>
      <ul>
        <li>Preservar e divulgar o folclore brasileiro em formato digital.</li>
        <li>Oferecer um recurso pedagógico acessível para professores e alunos.</li>
        <li>Demonstrar como tecnologias web podem ser aplicadas em soluções educativas interativas.</li>
      </ul>

<h2>Tecnologias Utilizadas</h2>
      <ul>
        <li><strong>HTML5</strong>: estruturação das páginas e conteúdo.</li>
        <li><strong>CSS</strong>: estilização e responsividade da interface.</li>
        <li><strong>JavaScript</strong>: lógica de busca e interação com a base de conhecimento.</li>
      </ul>

<h2>Funcionalidades</h2>
      <ul>
        <li>Busca por termos relacionados às lendas.</li>
        <li>Exibição de resultados com nome, região e descrição.</li>
        <li>Interface simples e intuitiva, pensada para uso em ambientes escolares.</li>
      </ul>


<h2>Público-Alvo</h2>
      <ul>
        <li><strong>Educadores</strong>: recurso para enriquecer aulas de cultura, literatura e história.</li>
        <li><strong>Estudantes</strong>: ferramenta de apoio para leitura e pesquisa.</li>
        <li><strong>Profissionais de tecnologia</strong>: exemplo prático de aplicação de linguagens web em projetos educativos.</li>
      </ul>
